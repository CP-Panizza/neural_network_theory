# 神经网络原理的探寻

#####引子
```text
    本人对于编程有着强烈的兴趣，对于人工智能非常的痴迷
    接触神经网络已有一年多的时间了，只知道神经网络的思想和算法不仅仅只是计算机编程，
它直接依赖于数学的思想来描述我们的世界。
从最开始接触神经网络的时候就被他的表达效果所惊讶，于是便立马搜寻了一些入门的教程进行学习和观看，
但是教程的内容起点较高，根本听不懂老师所讲的是什么东西，还用上了一些TensorFlow之类的科学计算库，
整个学习的积极性被严重打压。表面上只知道调用第三方库的某些方法可以快速的搭建一个网络并进行训练，
但是算法本身的原理确不知道怎么样去描述，连半瓶醋的水平都没有， 哎，真的佩服前人的智慧，是我等后人穷极一生也达不到的。
至此，便放弃了神经网络的学习。
    此后又经过了几个月的时间，不幸天灾人祸，2020年遇到了冠状病毒疫情，原本大三的我们只能在家通过网络学习，
我实在受够了这些老师的教学，在此时发现自己的时间比较充裕，便又打起了神经网络的主意。
```
